Servidor Web Simples - Laboratório 2 RCI

Bem-vindo!

Este é um servidor web simples implementado em Python para o Laboratório 2 da disciplina de Redes de Comunicação e Internet (RCI).

Informações do Servidor

  • Endereço: 192.168.56.21
  • Porta: 6789
  • Protocolo: HTTP/1.0

Como Testar

Método 1: Usando telnet

telnet 192.168.56.21 6789

Depois digite:

GET /index.html HTTP/1.0
Host: 192.168.56.21:6789

Método 2: Usando o cliente web Python

python webclient.py 192.168.56.21 6789 /index.html

Análise de Protocolos

Para analisar os pacotes TCP trocados, você pode usar:

  • Wireshark: Ferramenta gráfica para captura e análise de pacotes
  • tcpdump: Ferramenta de linha de comando para captura de pacotes

Estrutura da Requisição HTTP

GET /index.html HTTP/1.0
Host: 192.168.56.21:6789
Connection: close

Estrutura da Resposta HTTP

HTTP/1.0 200 OK
